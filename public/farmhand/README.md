# Farmhand sprites

Drop PNG files here to replace the emoji fallback.

Expected filenames (all optional — missing files fall back to emoji):

- `default.png` — idle / neutral
- `happy.png` — approving
- `thinking.png` — considering the user's answer
- `sassy.png` — roasting (the most common mood)
- `angry.png` — when the user is really cookin' the farmhand

**Recommended:** 128×128 or 256×256 PNG, transparent background, pixel-art style.

You can drop just one file (`default.png`) and it'll be used whenever that specific mood exists — every other mood keeps the emoji fallback until you add a file for it.
